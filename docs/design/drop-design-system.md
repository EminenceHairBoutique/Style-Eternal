## 1. Brand DNA — Extracted

### Core Identity

Style Eternal is a luxury streetwear label rooted in Newark, NJ — but the brand's emotional reach extends far beyond any single city. The design language is built on **emotional weight**, **title-led graphics**, **symbolic imagery**, and **premium garment construction**.

### Established Design Language

From the existing catalog (Drop 01: North Ward, Drop 02: Iron Bound, Drop 03: Love Never Dies, Essentials, Legacy Capsule), the following DNA has been identified and must be preserved:

| Element | Current Expression |
|---|---|
| **Title systems** | Large uppercase display lettering as dominant visual element — "Love Never Dies," "Forbidden Love," "Freedom for Minorities," "Eternal Love" |
| **Central figures** | Skeleton figures, silhouettes, embracing forms, raised fists |
| **Symbolic motifs** | Flames, hearts, broken hearts, handcuffs, crowns, stars, halos, speech bubbles |
| **Typography** | Mixed systems — block headlines + script overlays + condensed stacked captions + distressed lettering |
| **Color system** | Dark garment bases (washed black, charcoal, asphalt, concrete) with bright accent zones (red flames, gold stars, pink/magenta) |
| **Composition** | Layered collage — title + central symbol + secondary motifs + supporting text — compressed into a single print zone |
| **Print method** | Screenprint-first, distressed/cracked texture, vintage worn-in feel |
| **Garment weight** | Heavyweight — 280gsm tees, 400gsm hoodies, 300gsm long sleeves |
| **Fit** | Oversized primary, relaxed secondary |
| **Branding** | "Style Eternal," "SE," "EST. 2021" embedded in compositions — banners, crest systems, footer captions |

### Brand Position

- **Luxury streetwear with emotional weight**
- **Dark editorial luxury** web presence
- **Newark as origin story, not full identity**
- **Collectible, fashion-forward, wearable**

## 3. Design System Hierarchy

Every garment graphic follows this construction order:

```
1. TITLE          → largest visual element, the anchor
2. CENTRAL MOTIF  → one dominant symbol or figure
3. SUPPORT MOTIFS → 2–4 secondary symbols around the central figure
4. PLACEMENT      → hero print zone + supporting print zone
5. COMPOSITION    → symmetrical or near-symmetrical, layered fg/bg
```

### Composition Rules

- Title at top or as dominant anchor
- Central illustration or symbol below/behind title
- Secondary symbols compressed around the central figure
- Small supporting slogan or caption text at bottom or edges
- Optional sleeve continuation detail
- Text and image must overlap and interact — never fully separated

### Print Placement Logic

| Zone | Priority | Description |
|---|---|---|
| Back print | Primary | Large-format hero graphic |
| Front chest | Primary | Oversized centered graphic or left-chest brand mark |
| Sleeves | Secondary | Continuation graphics, flame runs, small symbols |
| Lower hem | Tertiary | Caption branding, "EST. 2021," supporting text |
| Side panel | Tertiary | Utility pieces only — tonal print hits |

**Rule:** Use one major print zone + one supporting zone. Avoid equally strong graphics across all zones on a single garment.

## 4. Color System

### Garment Base Palette

| Tier | Colors | Usage |
|---|---|---|
| Primary | Washed Black (#1A1A1A), Black (#0A0A0A), Charcoal (#3A3A3A) | Default for all graphic-heavy pieces |
| Secondary | Bone/Off-White (#E8E4DE), Muted Olive (#4A4A3A), Muted Brown (#6B5B4A) | Seasonal alternates, lighter pieces |
| Tertiary | Concrete Grey (#6B6B6B), Light Grey | Conceptually appropriate pieces only |

### Graphic Accent Palette

| Accent | Hex (approx.) | Usage |
|---|---|---|
| Bright Red | #CC2222 | Flames, hearts, statement accents |
| Orange / Flame | #E8661A | Fire, energy zones |
| Gold Yellow | #D4AF37 | Crowns, stars, brand accent |
| Pink / Magenta | #D94480 | Hearts, halos, love-themed accents |
| Purple / Lavender | #7B5EA7 | Smoke, depth, romantic contrast |
| Chrome / Silver | #C0C0C0 | Metallic hearts, restrained utility accents |
| Bone White | #F0EDE5 | Text, outlines, contrast on dark garments |

### Color Behavior Rules

- Keep garment dark overall
- Bright colors as focal zones — never full-garment saturation
- Most graphics: black + white + 1–3 strong accent colors
- Accent colors appear in isolated areas, not spread evenly
- No rainbow clutter, no soft pastel dominance

## 5. Typography System

### Primary Type Stack

| Role | Style | Usage |
|---|---|---|
| **Hero title** | Large uppercase display, distressed block | Largest element on garment — the anchor |
| **Script overlay** | Cursive/script, secondary emotional accent | Sits behind or below the title — not the main readability anchor |
| **Condensed stack** | Tight condensed uppercase, stacked | Supporting text, side captions, "EST. 2021" treatments |
| **Caption** | Small sans or condensed, bottom edges | "Style Eternal," "Born in Newark," collection references |

### Typography Rules

- Title is often the largest visual element
- Script is emotional accent, never primary readability
- Distressing built into the letterforms
- White/light outlines to separate colored type from dark garments
- Block headline + script subtitle combinations encouraged
- Type overlaps and integrates with artwork — not separated

### Anti-Drift

- No clean corporate sans-only systems
- No generic modern luxury text with no character
- No cartoon bubble type
- No unreadable blackletter overkill
- No more than 3 font styles per garment design

## 6. Illustration & Artwork Rules

### Visual Content Library

| Symbol | Drops | Notes |
|---|---|---|
| Skeleton figures | All | Core to brand DNA — simplify toward silhouette |
| Hearts / broken hearts | Eternal Love, Love Never Dies | Emotional anchor symbols |
| Flames | Eternal Struggle, Eternal Love | Red/orange for struggle, pink-red for love |
| Crowns / cracked crowns | Eternal Struggle | Power symbol — cracked = pressure |
| Scales / broken scales | Eternal Struggle | Justice symbol — broken = systemic failure |
| Fists | Eternal Struggle | Resistance — simplified, silhouette-driven |
| Halos / broken halos | Eternal Love | Sanctity + loss |
| Stars / starbursts | All | Brand accent, radiating energy |
| Smoke / smoke rings | Eternal Struggle, Utility | Atmosphere, controlled aggression |
| Chrome hearts | Eternal Love | Metallic love symbol |
| Surveillance eye (fragmented) | Eternal Struggle | Control symbol — abstract, not literal |
| Paired silhouettes | Eternal Love | Romantic unity |
| Laurel wreaths | Legacy, Utility | Heritage, achievement |
| Crosshair / target symbols | Eternal Struggle | Focused aggression |

### Construction Rules

- Build graphics as collages of multiple elements
- Flat graphic zones + distressed shading
- Bold and readable from distance
- Prioritize silhouette clarity
- Simplified enough to screenprint cleanly
- Strong central focal point

### Anti-Drift

- No photorealistic painting
- No overly soft painterly shading
- No abstract art with no symbolic clarity
- No tiny low-impact details that disappear in print
- No overly busy mural-style chaos

## 7. Texture & Print Finish

All artwork should feel physically printable and intentionally worn-in:

- Cracked ink look
- Vintage worn texture
- Washed garment appearance
- Scraped / weathered print edges
- Artificial aging in both text and artwork

The graphics should feel **broken-in, not clean-vector-flat**.

### Anti-Drift

- No glossy gradient-heavy digital art
- No airbrushed fantasy polish
- No overly clean print finishes unless conceptually justified

## 10. Drop Adaptation Rules

The same base design system applies across all drops, with symbol and accent shifts:

### Eternal Struggle

- Crowns, scales, surveillance fragments, silhouettes, fire, pressure
- Gold / red / orange accents
- Controlled aggression, symbolic resistance, power

### Eternal Love

- Hearts, halos, stars, broken halos, pink-red flames, paired silhouettes
- Pink / purple / chrome accents
- Romance after violence, loyalty after chaos, tenderness with edge

### Utility Capsule

- Reduced illustration density
- Title system and branding preserved
- Seam logic, technical marks, panel graphics, tonal print hits
- Minimal symbols — brand marks, small motifs, hardware emphasis

## 11. Quality Standard Checklist

Every concept must pass:

- [ ] Is the title visually dominant enough?
- [ ] Is the central symbol readable from distance?
- [ ] Does the piece feel collectible?
- [ ] Does the graphic feel distressed and screen-printable?
- [ ] Does the color system use bright accents intentionally?
- [ ] Does the design feel premium, not generic?
- [ ] Does it still connect to the current Style Eternal lineup?
- [ ] When in doubt, was composition simplified one step toward cleaner, more modern, more premium readability?
