# Drop Prompt Pack — OpenArt

> External art generation prompts for OpenArt AI
> Use these prompts to generate concept art, garment mockups, and graphic elements
> Always post-process with distressed/cracked textures and brand typography

---

## Usage Notes

1. These prompts generate **concept direction** — not final production art
2. Every generated image needs post-processing: add cracked ink textures, distressed edges, brand typography overlay
3. Use "negative prompt" fields to exclude: photorealistic skin, realistic faces, bright backgrounds, clean vector flat art
4. Generate at 1024×1024 minimum; 1536×1536 preferred for garment mockups
5. Model recommendation: Flux, SDXL, or Midjourney-style models for best results with streetwear concepts

---

## Global Style Modifiers

Add these to every prompt for consistency:

```
Style modifiers: luxury streetwear, dark editorial, premium fashion, distressed screen print texture, cracked ink effect, vintage worn aesthetic, high contrast, bold typography, collectible fashion design, heavyweight cotton garment, washed black fabric
```

```
Negative: photorealistic faces, bright cheerful colors, clean vector art, glossy digital finish, pastel palette, corporate branding, cartoon style, watercolor, soft painterly shading, rainbow colors
```

## POST-PROCESSING NOTES

After generating any image:

1. **Add cracked ink texture** — overlay distressed textures on all lettering and artwork
2. **Add garment context** — composite onto garment mockups if needed
3. **Adjust palette** — ensure colors match the brand palette (no neon, no pastel dominance)
4. **Check readability** — title must be readable from 10 feet away
5. **Add brand marks** — "Style Eternal," "SE," or "EST. 2021" must appear somewhere
6. **Verify distressing** — graphics should feel broken-in, not clean-vector-flat
