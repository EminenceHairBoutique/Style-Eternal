## TIPS FOR CANVA

### Recommended Post-Processing
1. Reduce brightness by 10-15% for darker mood
2. Add film grain overlay at 5-10% opacity
3. Apply subtle vignette
4. Desaturate slightly for muted tones
5. Add SE watermark/logo using Canva brand kit

### Color Codes for Brand Kit
- Black: `#0A0A0A`
- Charcoal: `#1A1A1A`
- Bone: `#E8E4DE`
- Gold: `#C4A35A`
- Red: `#C43030`

### Font Pairings in Canva
- Headlines: Oswald (Bold) — substitute: Antonio or Barlow Condensed
- Body: Inter — substitute: DM Sans
- Accent: Space Grotesk — substitute: Outfit or Urbanist
