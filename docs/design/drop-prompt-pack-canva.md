# Drop Prompt Pack — Canva

> Art generation prompts optimized for Canva's Magic Studio / Text to Image
> Use these to generate concept art, mood boards, and social media assets
> Canva's AI responds best to descriptive, natural-language prompts

---

## Usage Notes

1. Canva's AI models work best with **clear, descriptive sentences** — not technical parameters
2. Use "Styles" in Canva Magic Studio: select **"Photo"** or **"Graphic"** style depending on concept
3. Generate at the largest available resolution
4. Post-process every generated image: add brand typography, cracked textures, and garment context
5. These prompts are structured as natural-language descriptions that Canva handles well

## POST-PROCESSING CHECKLIST

After generating in Canva:

1. ☐ Apply cracked/distressed texture overlay on all lettering
2. ☐ Verify brand typography matches the Style Eternal type system
3. ☐ Check that accent colors match the brand palette — no neon, no pastel dominance
4. ☐ Ensure title text is readable from a distance
5. ☐ Add "Style Eternal," "SE," or "EST. 2021" if missing from generated art
6. ☐ Composite onto garment mockups using Canva's mockup tools
7. ☐ Export at maximum resolution for production use
